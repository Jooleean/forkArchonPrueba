# Rancho

Este proyecto es Rancho, un videojuego altamente inspirado en el clásico Archon: The Light and the Dark.
Adéntrate en un rancho, donde los animales batallarán para encontrar al lado ganador.
Explota huevos, lanza púas y controla el clima para derrotar al rival, todo está en tus manos.
